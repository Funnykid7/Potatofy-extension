What it actually does.

Six things, all on by default. You can flip any of them off, and you can whitelist any site with one click.

Blocks trackers, ads & web fonts before they load.

Network-layer blocking via Chrome's declarative rules, no per-request CPU cost. Around 250 rules cover the usual analytics, ad networks, and font CDNs.

Discards idle tabs.

If you haven't touched a tab in a few minutes, it goes to sleep. The tab stays in your strip, the memory does not.

Freezes background JavaScript.

When a tab is hidden, its timers and animation frames stop running. Switch back and they pick up exactly where they were.

Pauses background video.

The moment a tab loses focus, any playing video is paused and its preload is dropped. The decoder gives the memory back.

Strips heavy CSS animations and prefetch links.

No more autoplaying scroll parallax draining your GPU. No more pages preloading 40 things you didn't ask for.

Whitelist any site with one click.

Banking, video calls, sites that genuinely need everything to run. Open the popup, hit allow, done.

Install in about a minute.

There is no Web Store listing yet. The whole repo is the extension, no build step, no node_modules.

  1. 1

    Clone the repo.

    git clone https://github.com/Funnykid7/Potatofy-extension.git
  2. 2

    Open chrome://extensions and turn on Developer mode.

    The toggle is in the top-right corner. Same flow works on Chromium, Brave, Edge, and Arc.

  3. 3

    Click Load unpacked and pick the cloned folder.

    You should see a small potato icon land in your toolbar. That is the popup. Click it to toggle features or whitelist the current site.

Common questions.

Will it break sites?

Most sites are fine. Heavy single-page apps that rely on background timers can act weird while in a background tab, but they recover the moment you switch back. If a site genuinely needs everything, one click whitelists it.

Why Raspberry Pi specifically?

Because the Pi is the loudest example of "modern hardware that should be enough but isn't, because the web is heavier than it should be." Potatofy works on any low-spec machine: old laptops, Chromebooks, mini PCs. The Pi is just where the difference is most dramatic.

Does it phone home?

No. There is no analytics, no telemetry, no remote rule fetch. The whole rule list is in the repo. Your settings sync to your Chrome profile through Chrome's own storage, not through any server we run, because we do not run any.

Is it free?

Yes, and the source is GPLv3. Read it, change it, fork it.

Can I trust the numbers?

Numbers in this page come from a real Pi 5 with Chromium's built-in memory tracing, comparing the same set of tabs with and without the extension. The popup's own counter is an estimate based on tuned weights per event, not a kernel-level measurement. It is meant to show direction, not lab precision.
